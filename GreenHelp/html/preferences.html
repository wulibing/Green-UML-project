Preferences

Generate stubs for constructors from superclass
      If selected, the option to create method stubs to call the superclass's constructor will automatically be created when a class is created from within Green's editor.

Save compilation units automatically after code modification
      If selected, whenever Green's editor gains focus, all other open editors will be saved so that Green's diagram will always be live with the code.

Show fully-qualified type names
      Toggles the display for fully-qualified names in field declarations, method signatures, and the title of UML class boxes.

Show method parameter names
      If selected, the names of method parameters will be shown. If not selected, only the declared type of the parameters will be shown.

Show non-public methods
      If selected, all methods inside a UML class box will be visible. If not selected, only public methods will be visible.

Relationship line width
      Changes the width of the lines on relationship arcs.

Font (under "Colors and Fonts")
      Changes the font used to display captions in UML boxes and notes. Note that the setting the font's style (bold/italics/underline, etc.) will have no effect on Green's display.
